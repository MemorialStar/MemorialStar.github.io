나전칠기의 빛

포인터를 움직여 보세요 — 광원의 각도가 바뀌며
실제 자개처럼 간섭색이 흐릅니다.

▼ 아래에 원리와 사용법

구현된 광학 원리 장식이 아니라 계산입니다 — 픽셀마다 스펙트럼을 적분합니다

① 다층 박막 간섭 아라고나이트 판(n≈1.63, ~470nm)과 유기층(n≈1.43)이 쌓인 구조. 광로차 OPD = 2(n₁d₁cosθ₁ + n₂d₂cosθ₂)를 계산해 파장별 보강/상쇄를 구합니다.
② 브래그 스택 반사 N층 반사파의 합 sin²(Nφ/2)/(N²sin²(φ/2)) — 층수가 많을수록 피크가 날카로워져 채도가 올라갑니다.
③ 스펙트럼 → 색 400–700nm를 샘플링해 CIE 1931 등색함수로 적분 → XYZ → sRGB. 팔레트가 아니라 물리에서 색이 나옵니다.
④ 각도 의존 색변화 기울이면 cosθ가 커져 OPD가 줄고 색이 단파장(파랑) 쪽으로 이동 — 포인터/기기 기울임이 광원각을 제어합니다.
⑤ 판상 구조와 결 보로노이 셀 = 자개 판 도메인, 저주파 노이즈 = 성장 결. 판마다 두께·기울기가 달라 조각조각 다른 색을 냅니다.
⑥ 테라스 프린지와 글린트 판 경계의 두께 계단이 만드는 간섭 프린지, 일부 판만 정반사각에서 번쩍이는 글린트 — 자개 특유의 반짝임입니다.

강조 패널 Nacre.attach() — 카드·버튼·섹션 배경으로

프로젝트 하이라이트

포트폴리오에서 강조하고 싶은 카드에 이렇게 배경으로 깔 수 있습니다. 패널마다 seed·두께를 다르게 주면 서로 다른 자개 조각처럼 보입니다.

염색 시트 스타일 style: 'sheet' — 밝게 염색된 자개 (sample.png 재현)

어두운 옻칠 상감 대신 밝게 염색된 자개의 결입니다. 결의 방향장이 공간에서 완만하게 휘고(성장선의 곡률), 층수를 낮춘 파스텔 간섭색과 흰 진주광택이 얹힙니다. 아래 슬라이더로 실시간 조정해 보세요.

텍스트 하이라이트 Nacre.text() — WebGL 없이 같은 물리식으로 그라디언트 생성

본문 속에서 이렇게 중요한 문구만 자개처럼 빛나게 하거나,
두께를 바꿔 다른 색조를 낼 수도 있습니다.

파라미터 배경 인스턴스에 실시간 반영 — 마음에 드는 값을 포폴에 쓰세요

사용법

<script src="nacre/nacre.js"></script>
<script>
  // 배경 / 강조 패널 (WebGL) — 기본은 옻칠(lacquer) 스타일
  const bg = Nacre.attach('#hero', { scale: 6.5, thickness: 470, layers: 6 });
  bg.setOptions({ thickness: 520 });   // 실시간 변경

  // 염색 시트 스타일 — tint(염색색)·flow(결 방향)·streak(결 밀도) 등 조정 가능
  Nacre.attach('#card', { style: 'sheet', tint: '#4a86d8', flow: -30, streak: 1.2 });

  // 텍스트 하이라이트 (WebGL 불필요) — tint를 주면 시트 톤
  Nacre.text('.nacre-text', { thickness: 470 });
  Nacre.text('#sheet-text', { tint: '#4a86d8' });

  // 간섭색 하나만 필요할 때 (링크·보더 색 등)
  el.style.borderColor = Nacre.color(470, 0.8, 6);

  // 또는 HTML만으로: <div data-nacre data-thickness="430"> + Nacre.auto()
</script>
